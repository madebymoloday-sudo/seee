# Переделка дизайна страницы "Безопасность"

## Текущие проблемы

1. Неструктурированный дизайн
2. Непоследовательное оформление элементов
3. Отсутствие визуальной иерархии
4. Неудобное расположение кнопок

## Новый дизайн

### Структура:
1. **Электронная почта** - карточка с иконкой, описанием, полем ввода и кнопкой
2. **Двухфакторная аутентификация** - карточка с иконкой, статусом (badge), описанием и кнопкой
3. **Служба поддержки** - карточка с иконкой, описанием и кнопкой

### Стили:
- Карточки с тенями и скругленными углами
- Иконки для каждого раздела
- Цветные badges для статусов
- Кнопки с брендовым цветом
- Адаптивный дизайн
