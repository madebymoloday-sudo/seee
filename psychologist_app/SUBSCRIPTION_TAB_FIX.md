# Исправление отображения вкладки "Подписка"

## Проблема

Вкладка "Подписка" не отображается в личном кабинете.

## Решение

### 1. JavaScript автоматически создает вкладку

Функция `ensureSubscriptionTab()` автоматически создает вкладку при открытии личного кабинета.

### 2. Если вкладка все еще не появляется

Проверьте HTML шаблон личного кабинета и добавьте вручную:

**В контейнер с кнопками вкладок:**
```html
<button class="tab-btn" data-tab="subscription">Подписка</button>
```

**В контейнер с контентом вкладок:**
```html
<div id="tab-subscription" class="tab-content"></div>
```

### 3. Расположение вкладки

Вкладка должна быть:
- После "Безопасность"
- Перед "Настройки"

---

## Проверка

1. Откройте личный кабинет
2. Проверьте, есть ли кнопка "Подписка" среди вкладок
3. Если нет - проверьте консоль браузера на ошибки
4. Убедитесь, что функция `ensureSubscriptionTab()` вызывается

---

## Отладка

Добавьте в консоль браузера:
```javascript
// Проверить, есть ли модальное окно
console.log(document.getElementById('cabinetModal'));

// Проверить, есть ли вкладки
console.log(document.querySelectorAll('.tab-btn'));

// Вызвать функцию вручную
ensureSubscriptionTab();
```
